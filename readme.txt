Autoblog WPMagis is a WordPress Plugin for automatically publishing posts from a specific folder.

== Description ==

Autoblog WPMagic Plugin for WordPress makes it possible to automate the task of publishing blog posts. 
The plugin searches for files in a specific folder and publishes content in a blog post. 

== Installation ==
* Install it from Plugin Menu -> Add new.
* Upload your .txt files to the folder /wp-content/uploads/autoblog-wpm-free/.
* Check a <a href="https://wpmagic.cloud/wp-content/uploads/autoblog-wpm-free/3_Must_Do_Steps_To_A_Happier_Healthier_Day.txt">sample file</a> that shows you how the .txt files should be formatted.
* Wait for the file's table to populate or manually populate the table with files in the "Settings".
* The plugin will publish periodically content from the files as blog posts on your website.
